# Table - Syntax

## Align Columns

### Default

By default, column titles are center-aligned, and column contents are left-aligned.

Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |
```

Output

| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |

### Left-Aligned Columns

Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |
```

Output

| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |

### Right-Aligned Columns

Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |
```

Output

| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |

### Center-Aligned Columns

Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
```

Output

| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
