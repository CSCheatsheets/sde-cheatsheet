# `tmux` - Command

## Show All Active Sessions

```shell
tmux ls
```

## Create a New Session

```shell
tmux new -s [session_name]
```

## Attach to an Existing Session

```shell
tmux attach -t [session_name]
```

## Split the Window inside a Session

```shell
ctrl + b + "
```

## Exit a Session without Termination

```shell
ctrl + b + d
```

## Terminate a Session

```shell
ctrl + d
```
